# Assets/_Project

Esta carpeta contiene únicamente contenido creado o adaptado específicamente para
Cartridge & Cloud.

- `Art`: materiales, modelos, sprites, texturas y VFX.
- `Audio`: música y efectos.
- `Content`: localización y datos estáticos.
- `Prefabs`: prefabs propios.
- `Scenes`: escenas del juego y laboratorios de prueba.
- `Scripts`: código organizado por responsabilidad arquitectónica.
- `Settings`: configuraciones propias del proyecto.
- `Tests`: pruebas EditMode y PlayMode.
- `UI`: fuentes, iconos, estilos y vistas.

Los paquetes de Unity permanecen en `Packages`. Los assets externos futuros deberán
ubicarse en una raíz separada y conservar sus licencias.
