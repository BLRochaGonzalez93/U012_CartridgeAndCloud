# 1. Posicionamiento

Simulador acogedor y profundo de tienda de videojuegos, centrado en construcción funcional, logística visible y progresión empresarial.

# 2. Materiales futuros

- key art;
- cápsulas Steam;
- trailer del vertical slice;
- GIFs de construcción, recepción, clientes y caja;
- devlogs técnicos;
- screenshots comparativas de progresión.

# 3. Gates

No iniciar campaña pública fuerte antes de aprobar H6. El contenido técnico actual sirve para portfolio y seguimiento, no como promesa de producto final.

# 4. Mensajes

- diseña tu tienda;
- gestiona stock y clientes;
- crece desde un pequeño local;
- convierte decisiones espaciales en resultados comerciales.
