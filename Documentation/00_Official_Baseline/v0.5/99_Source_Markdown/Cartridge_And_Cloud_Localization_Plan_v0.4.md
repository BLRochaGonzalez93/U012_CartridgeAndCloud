# 1. Idiomas

- Español como idioma de producción.
- Inglés como idioma comercial mínimo para Steam.
- Otros idiomas según viabilidad posterior.

# 2. Reglas

- Ningún texto jugable final hardcodeado.
- IDs estables y tablas por contexto.
- Variables y plurales preparados.
- UI diseñada para expansión de texto.
- Validación lingüística antes de release.

# 3. Estado

El paquete Localization está instalado. La integración de contenido localizado se realizará con UI/UX del Sprint 15.
