# 1. Regla de seguridad

Ningún recurso entra en build pública, Steam, trailer, web o press kit sin origen, licencia, uso comercial, atribución y evidencia registrados.

# 2. Categorías

Arte, audio, fuentes, plugins, paquetes Unity, código externo, IA generativa, traducción, colaboración, testing, marketing y documentación.

# 3. Estados

Aprobado, Pendiente, Temporal, Reemplazar, Prohibido y Retirado.

# 4. Campos del registro

ID, recurso, categoría, autor/proveedor, fuente, versión, licencia, uso permitido, modificación, redistribución, atribución, evidencia, fecha, ubicación, estado, riesgo y responsable.

# 5. Política de IA generativa

Registrar herramienta, fecha, prompt/resumen, edición humana, destino, términos aplicables y revisión de similitud. No usar como prueba de propiedad exclusiva ni imitar deliberadamente marcas o artistas vivos.

# 6. Marcas y nombres

`Cartridge & Cloud` es provisional. Antes de Steam, dominios o campañas debe realizarse búsqueda de marca, nombre comercial, redes y posibles conflictos. Los cinco candidatos históricos se conservan internamente.

# 7. Marcas ficticias

Productos, plataformas, transportistas y competidores deben usar nombres propios revisados para evitar confusión con empresas reales.

# 8. Colaboradores

Toda aportación que entre en producto o marketing requiere autorización escrita sobre uso comercial, modificación, crédito y compensación si procede.

# 9. Fuentes

No redistribuir fuentes salvo licencia expresa. Registrar archivos, versión y alcance. Preferir licencias OFL o equivalentes revisadas.

# 10. Open source y plugins

Conservar LICENSE/NOTICE, versión, repositorio, modificaciones y obligaciones. Revisar compatibilidad antes de actualizar.

# 11. Créditos

Secciones previstas: dirección/desarrollo, arte, audio, QA, traducción, herramientas, software de terceros, assets y agradecimientos.

# 12. Checklist público

- [ ] Nombre validado.

- [ ] Textos de atribución incluidos.
- [ ] Música y trailer con licencia comercial.
- [ ] Capturas sin contenido de terceros no autorizado.
