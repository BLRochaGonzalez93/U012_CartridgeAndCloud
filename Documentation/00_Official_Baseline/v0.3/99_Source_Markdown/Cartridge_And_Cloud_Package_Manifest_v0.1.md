## 1. Propósito

Este manifiesto enumera los entregables del paquete documental reconstruido. La documentación sustituye el enfoque anterior de *Indie Studio Simulator*, parte desde Sprint 0 y usa `Enfoque_v0.6.md` como fuente conceptual aprobada. No se atribuye implementación a ningún sistema, escena, manager, build o sprint anterior.

- **Conceptualmente aprobado:** definido por `Enfoque_v0.6.md`, pero todavía no implementado.

- **MVP / Vertical Slice:** alcance de producción, no evidencia de una build existente.
- **Steam previsto:** intención de publicación; Steamworks no está iniciado ni integrado.
- **Provisional:** sujeto a prototipo, balance, disponibilidad comercial o validación legal.
