# Cartridge & Cloud

[English](README.en.md) | [Español](README.md)

## Resumen

**Cartridge & Cloud** es el título provisional de un simulador 3D de gestión empresarial desarrollado por **VRM Games** para PC/Steam. El jugador comienza operando físicamente una pequeña tienda de videojuegos y tecnología y puede hacerla evolucionar hacia comercio online, publishing, desarrollo interno y una plataforma digital propia.

## Estado

**Preproducción completa a nivel conceptual. Desarrollo reiniciado desde cero.**

No se considera implementado ningún sistema, escena, manager, build o sprint anterior. La fuente conceptual vigente es `Enfoque_v0.6.md` y el primer objetivo de producción es un vertical slice de la tienda física.

## Primer vertical slice

Incluye movimiento y cámara, construcción sobre cuadrícula, mobiliario, proveedor, pedidos, recepción, almacén, reposición, precios, clientes, cola, cobro, apertura/cierre, economía, resumen y guardado/carga.

No incluye empleados completos, investigación funcional, comercio online, publishing, desarrollo interno, plataforma ni servidores.

## Tecnología prevista

- Unity 6 LTS
- C#
- Universal Render Pipeline
- Input System
- AI Navigation / NavMesh
- Canvas/uGUI y TextMeshPro
- Unity Test Framework
- PC / Windows x64
- Steam como publicación prevista

## Documentación

El paquete documental contiene GDD, TDD, modelo de datos, UX/UI, biblias de arte/audio, QA, roadmap desde Sprint 0, Exceles de producción y planes legal, localización, marketing y Steam.

## Nombre provisional

El nombre comercial se validará antes de crear materiales públicos definitivos o iniciar Steamworks. No se han renombrado repositorios o namespaces históricos como parte de esta documentación.
