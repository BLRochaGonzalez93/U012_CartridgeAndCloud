# Cartridge & Cloud

[English](README.en.md) | [Español](README.md)

## Overview

**Cartridge & Cloud** is the provisional title of a 3D business-management simulator developed by **VRM Games** for PC/Steam. The player starts by physically operating a small videogame and technology store and can grow it into an online retailer, publisher, internal development studio and digital platform.

## Status

**Conceptual pre-production complete. Development restarted from scratch.**

No previous system, scene, manager, build or sprint is considered implemented. `Enfoque_v0.6.md` is the current conceptual source of truth, and the first production target is a vertical slice focused on the physical store.

## First vertical slice

It includes movement and camera controls, grid-based construction, furniture, supplier orders, receiving, warehouse inventory, restocking, pricing, customers, queues, checkout, opening/closing, economy, daily reports and save/load.

It excludes full employees, functional research, online commerce, publishing, internal development, the digital platform and server infrastructure.

## Planned technology

- Unity 6 LTS
- C#
- Universal Render Pipeline
- Input System
- AI Navigation / NavMesh
- Canvas/uGUI and TextMeshPro
- Unity Test Framework
- PC / Windows x64
- Steam as the intended release platform
