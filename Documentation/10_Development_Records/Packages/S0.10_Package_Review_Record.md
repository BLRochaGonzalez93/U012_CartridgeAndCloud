## Procedimiento

Para cada paquete aprobado para retirada:

1. Cerrar Test Runner.
2. Abrir Package Manager.
3. Retirar un único paquete.
4. Esperar a que Unity compile.
5. Confirmar consola limpia.
6. Ejecutar EditMode 5/5.
7. Ejecutar PlayMode 4/4.
8. Registrar PASS o restaurar el paquete.
9. Continuar con el siguiente.

No editar manualmente `Packages/manifest.json`.

## Cierre

- [ ] Todas las decisiones están registradas.
- [ ] Cada retirada tiene validación 9/9.
- [ ] `manifest.json` y `packages-lock.json` son coherentes.
- [ ] No quedan paquetes en estado `Review`.
