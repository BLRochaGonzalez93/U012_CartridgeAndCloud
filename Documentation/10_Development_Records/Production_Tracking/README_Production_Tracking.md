# Production Tracking — Operational Layer

## Propósito

Estos libros reflejan el estado real de producción posterior a la línea base
oficial v0.3.

La línea base oficial permanece en:

```text
Documentation/00_Official_Baseline/v0.3
```

No se modifica ni se sustituye.

## Libros

`Cartridge_And_Cloud_Production_Tracking_Operational_v0.3.1.xlsx`

## Política de actualización

1. Actualizar los libros al cerrar cada bloque o sprint.
2. No declarar PASS sin validación del desarrollador.
3. Registrar deuda y observaciones de forma explícita.
4. Usar exclusivamente los tipos de trazabilidad permitidos.
5. No editar retrospectivamente la línea base oficial.
6. Cuando una nueva versión oficial sustituya v0.3, archivarla mediante un
   proceso documental explícito.
