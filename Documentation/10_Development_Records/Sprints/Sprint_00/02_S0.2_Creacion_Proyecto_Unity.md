# S0.2 — Creación limpia del proyecto Unity

## Resultado esperado

Proyecto `CartridgeAndCloud` creado desde la plantilla Universal 3D de Unity 6.3 LTS, sin contenido histórico y con la configuración mínima consolidada.

## 1. Crear el proyecto

1. Abrir Unity Hub.
2. Ir a `Projects`.
3. Pulsar `New project`.
4. Seleccionar Editor `6000.3.18f1`.
5. Seleccionar la plantilla:
   - `Universal 3D`, o
   - la plantilla URP 3D equivalente mostrada por el Hub.
6. Configurar:
   - Project name: `CartridgeAndCloud`
   - Location: ruta local elegida.
7. Desactivar la conexión automática a servicios Cloud si el Hub ofrece esa opción.
8. Crear el proyecto.

## 2. Primera importación

1. No interactuar con carpetas del proyecto mientras Unity importa.
2. Esperar a que desaparezca la actividad de compilación/importación.
3. Abrir `Window > General > Console`.
4. Activar temporalmente:
   - Collapse.
   - Clear on Play.
5. Pulsar `Clear`.
6. Esperar unos segundos.
7. Confirmar:
   - 0 errores rojos;
   - no hay diálogos de migración;
   - no hay referencias a proyectos anteriores.

Si hay errores, no instalar paquetes adicionales hasta resolverlos.

## 3. Confirmar versión real del proyecto

Abrir el archivo:

```text
ProjectSettings/ProjectVersion.txt
```

Debe contener una línea equivalente a:

```text
m_EditorVersion: 6000.3.18f1
```

No editar ese archivo manualmente.

## 4. Verificar URP

1. Abrir `Edit > Project Settings > Graphics`.
2. Confirmar que existe un Render Pipeline Asset asignado.
3. Abrir `Edit > Project Settings > Quality`.
4. Confirmar que los niveles de calidad utilizan assets URP válidos.
5. Seleccionar una cámara de la escena de plantilla y comprobar que usa componentes compatibles con URP.

No convertir desde Built-in Render Pipeline. El proyecto debe nacer directamente como URP.

## 5. Configurar identidad del producto

Abrir:

```text
Edit > Project Settings > Player
```

Configurar:

| Campo | Valor |
|---|---|
| Company Name | `VRM Games` |
| Product Name | `Cartridge & Cloud` |
| Version | `0.0.1` |
| Default Icon | Sin definir en este bloque |

Configurar el identificador de Windows cuando el campo esté disponible:

```text
com.vrmgames.cartridgeandcloud
```

No utilizar espacios ni mayúsculas en el identificador.

## 6. Configurar renderizado base

En Player Settings:

```text
Other Settings > Rendering > Color Space
```

Seleccionar:

```text
Linear
```

No modificar todavía:

- Render Scale.
- HDR.
- MSAA.
- Shadow Distance.
- Dynamic Resolution.
- Render Graph.
- Shader stripping.

Esos valores se perfilarán cuando existan escenas representativas.

## 7. Configurar serialización y metadatos

Abrir:

```text
Edit > Project Settings > Editor
```

Configurar:

| Ajuste | Valor |
|---|---|
| Version Control Mode | `Visible Meta Files` |
| Asset Serialization Mode | `Force Text` |

Si Unity 6.3 muestra Version Control en una categoría separada, usar esa categoría para `Visible Meta Files`.

## 8. Configurar IDE

Abrir:

```text
Edit > Preferences > External Tools
```

Seleccionar el IDE instalado.

Regenerar archivos de proyecto únicamente si la integración no los genera de forma automática.

## 9. Configuración de entrada

No cambiar `Active Input Handling` antes de instalar Input System.

Se configurará después en S0.3 como:

```text
Input System Package (New)
```

No se utilizará `Both` como configuración definitiva.

## 10. Servicios y conectividad

No inicializar:

- Analytics.
- Ads.
- Cloud Save.
- Authentication.
- Remote Config.
- Multiplayer.
- Version Control de Unity.
- DevOps.
- Collaborate.

El control de versiones será Git y se configurará en S0.4.

## 11. Limpiar contenido de ejemplo

La plantilla puede incluir una escena o assets de muestra.

En este bloque:

- se pueden conservar temporalmente hasta confirmar que URP funciona;
- no deben confundirse con contenido del juego;
- no deben renombrarse todavía como escenas definitivas;
- se retirarán o reemplazarán al crear las escenas base del Sprint 0.

No importar Starter Assets ni samples de paquetes.

## 12. Guardar y cerrar

1. Guardar la escena actual.
2. Ejecutar `File > Save Project`.
3. Cerrar Unity.
4. Abrir de nuevo desde Unity Hub.
5. Confirmar una segunda importación limpia.
6. Comprobar de nuevo la consola.

- [ ] Plantilla Universal 3D / URP.
- [ ] 0 errores de compilación.
- [ ] URP Asset asignado.
- [ ] Company Name correcto.
- [ ] Product Name correcto.
- [ ] Version `0.0.1`.
- [ ] Identificador correcto.
- [ ] Color Space `Linear`.
- [ ] Visible Meta Files.

- [ ] IDE seleccionado.

- [ ] No existe contenido histórico.
