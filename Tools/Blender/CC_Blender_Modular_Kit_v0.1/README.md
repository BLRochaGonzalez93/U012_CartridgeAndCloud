# Cartridge & Cloud - Blender Modular Kit v0.1

Procedural Blender Python starter kit for non-character assets.

## Scope
- Architecture A01-A10
- Furniture F01-F08
- Products P01-P06
- Modular props for eight future expansion areas
- Shared preview materials
- Basic LOD and box-collider generation
- Validation report and Unity FBX batch export

Characters, rigging and animation are intentionally excluded.

## Fast start
1. Open Blender 4.x.
2. Open `scripts/build_all.py` in the Scripting workspace.
3. Press **Run Script**.
4. Review generated collections and geometry.
5. Run `validate_scene.py`.
6. Save the `.blend` file.
7. Run `export_unity_fbx.py` only after visual and functional review.

The generated geometry is a production-oriented parametric starting point, not final polished art.
